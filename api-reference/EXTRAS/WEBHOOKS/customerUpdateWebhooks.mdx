## Envelope

<ResponseField name="type" type="string">
Unique event type. Check documentation below to see available types.

_Example:_ `app.amlyze.customer.riskmanagement.kyc_scoring.completed`
</ResponseField>

<ResponseField name="specversion" type="string">
[Cloud Events](https://cloudevents.io/) Specification version that is being used.

_Example:_ `1.0`
</ResponseField>

<ResponseField name="source" type="URI-reference">
[RFC 3986 - Uniform Recource Identifier (URI)](https://datatracker.ietf.org/doc/html/rfc3986#section-4.1)

Event source.

_Example:_ `customer`
</ResponseField>

<ResponseField name="subject" type="string">
Event subject represents a resource external id.

_Example:_ `82deaa92-d559-40bd-96b7-053341a08e28`
</ResponseField>

<ResponseField name="id" type="string">
Unique event ID.

_Example:_ `0412a5ca-3be7-46b7-87de-38366397602e`
</ResponseField>

<ResponseField name="time" type="date">
Event time.

_Example:_ `2019-12-13T12:01:02.694Z`
</ResponseField>

<ResponseField name="datacontenttype" type="string">
Data content type

_Example:_ `application/json`
</ResponseField>

<ResponseField name="data" type="object">
Event response. Differs on the event type.
</ResponseField>

> ### com.amlyze.customer.updated

<ResponseField name="customerExternalID" type="string">
Customer external ID.

_Example:_ `82deaa92-d559-40bd-96b7-053341a08e28`
</ResponseField>

<ResponseField name="sanctioned" type="boolean">
Customer sanctioned or not.
</ResponseField>

<ResponseField name="pep" type="boolean">
The customer is/isn't a politically exposed person.
</ResponseField>

<ResponseField name="inAdverseMedia" type="boolean">
Customer found in adverse media or not.
</ResponseField>

<ResponseField name="inInternalList" type="boolean">
Customer found in internal list or not.
</ResponseField>

<ResponseField name="riskLevel" type="string from enum">
Customer risk level.

Enum class values: `NONE`,  `LOW`, `MEDIUM`, `HIGH`, `EXTREME`
</ResponseField>

<ResponseField name="status" type="string from enum">
Customer risk level.

Enum class values: `ACTIVE`,  `PENDING`, `REJECTED`, `SUSPENDED`, `CLOSED`
</ResponseField>

<ResponseField name="updatedAt" type="date">
Customer status update time in Amlyze system.

_Example:_ `2019-12-13T12:01:02.694Z`
</ResponseField>

_Example:_ 
```json
{
  "customerExternalID": "82deaa92-d559-40bd-96b7-053341a08e28",
  "sanctioned": true,
  "pep": true,
  "inAdverseMedia":true,
  "inInternalList":true,
  "riskLevel": "EXTREME",
  "status": "PENDING",
  "updateAt": "2019-12-13T12:01:02.694Z"
}
```
